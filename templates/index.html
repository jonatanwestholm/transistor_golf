Components from the top. Positive transistor (connects green tiles when red tile is high), Negative transistor (connects green tiles when blue tile is low), Bar (connects tiles), VDD (voltage supply, always high), GND (ground, always low), X (inputs), T (output), Isolator (stops overlapping bars from connecting)

Purpose. Connect inputs and output to each other and to transistors in order to make only the desired input-output relationships feasible. The only rule is that if two tiles are connected to each other, then they have the same voltage (high or low). To get started, you can implement the circuit described here
